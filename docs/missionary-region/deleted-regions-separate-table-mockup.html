설계 의도 (v2.1)

필터 카드 내부 이동 이유 각 테이블 카드 안에 검색/필터가 있으면 "이 필터가 어느 테이블에 적용되는가"가 구조적으로 명확. 레이아웃이 필터-테이블 관계를 직접 표현.
페이지 헤더 단순화 외부 필터 제거로 페이지 헤더는 타이틀만. 등록 버튼은 카드 헤더 우측으로 이동 → 버튼과 데이터가 같은 카드 안에 공존.
삭제됨 카드 필터 위치 필터는 패널 내부에 배치 — 닫혀있을 때는 보이지 않음. 차수 필터는 삭제됨 컨텍스트에서 덜 중요하므로 선교 그룹만 제공.
인라인 필터 스타일 외부 필터보다 작은 height(34px) + 배경(#FAFAFA) — 카드 본문과 시각적 구분은 유지하면서 과하지 않게.
